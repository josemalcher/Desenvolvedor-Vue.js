# Vue 2.0 com VueX

https://www.schoolofnet.com/curso/frontend/vuejs/vue-20-com-vuex/

O Vuex é uma biblioteca que trabalha com manutenção de estado da aplicação implementando os conceitos da Fonte Única da Verdade. Ele faz parte do mundo Vue.js e é essencial para construir aplicações front-end baseadas em componentes. Neste curso iremos conhecer os novos conceitos principais da versão 2.0. ATUALIZAÇÃO 2
